# IT-company Task Manager

## Description

This project is a Django-based web application for task management for an IT company.
It allows managing workers, tasks, task types, and positions.

## Features

* Worker management
* Task assignment
* Filtering and search
* Authentication system

## Installation

```bash
git clone <your-repo-link>
cd your-project
python -m venv venv
venv\Scripts\activate  # Windows
pip install -r requirements.txt
```

## Run project

```bash
python manage.py migrate
python manage.py runserver
```
